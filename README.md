### 如何安装与使用
> * git clone  https://github.com/ayuhuajin/base-vue-instance.git  &emsp;// 下载 
> * cd base-vue-instance //进入 base-vue-instance目录
> * npm install //安装依赖
> * npm run serve //项目运行
> * npm run build  //项目打包
> * npm run tests  //测试（暂无）
> * npm run lint  //格式化


## 技术栈

- Vue-Cli（Vue 脚手架工具）
- Vue（核心框架）
- Vue-Router（页面路由）
- Vuex（状态管理）
- ES 6 / 7 （JavaScript 语言的下一代标准）
- Scss（CSS预处理器）
- Axios（网络请求）
